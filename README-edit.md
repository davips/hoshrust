## Python installation
### from package
```bash
# Set up a virtualenv. 
python3 -m venv venv
source venv/bin/activate

# Install from PyPI
pip install hosh
```

### from source
```bash
cd my-project
git clone https://github.com/davips/hosh ../hosh
pip install -e ../hosh
```
