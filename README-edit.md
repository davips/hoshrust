# halg
Fast cryptographic hash and algebra for Rust and Python.

## Python installation
### from package
```bash
# Set up a virtualenv. 
python3 -m venv venv
source venv/bin/activate

# Install from PyPI
pip install halgpy
```

### from source
```bash
cd my-project
git clone https://github.com/davips/halg ../halgpy
pip install -e ../halgpy
```
