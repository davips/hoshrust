# osh
Fast cryptographic hash (half-blake3) and operators for Rust and Python.

## Python installation
### from package
```bash
# Set up a virtualenv. 
python3 -m venv venv
source venv/bin/activate

# Install from PyPI
pip install osh
```

### from source
```bash
cd my-project
git clone https://github.com/davips/osh ../osh
pip install -e ../osh
```
